- `ada:VLMImage`
- `ada:VLMImageCollection`

## Valid Component Types
- `ada:VLMImage`
- `ada:VLMImageCollection`
- `ada:analysisLocation`
- `ada:methodDescription`
- `ada:instrumentMetadata`
